# dim2xdf
DIM image file to XDF image file converter with Python

### Install

    pip install git+https://github.com/tantanGH/dim2xdf.git

[Windowsユーザ向けPython導入ガイド](https://github.com/tantanGH/distribution/blob/main/windows_python_for_x68k.md)

### Usage

    dim2xdf <dim-file> <xdf-file>

Note that you need to add dim2xdf installed folder to your PATH environment variable.
